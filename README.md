# 🎯 VR 활쏘기 미니게임 프로젝트

## 📌 분야 선택
- **XR 분야 중: VR(Virtual Reality)** 선택

## 🧠 목표 기능
Meta Quest 2 기반으로 작동하는 **핸드 트래킹 or 컨트롤러 기반의 활쏘기 미니게임** 구현

---

## 🛠️ 주요 구현 기능

| 기능 | 설명 |
|------|------|
| 🏹 활과 화살 시스템 | 손으로 당기고 발사하는 리얼한 활쏘기 구현  
| 🎯 과녁 점수 | 실제과녁과 비슷하게 타격지점을 계산해서 점수를 추가
| 🌬️ 바람 시스템 | 실행시 1회 설정
| 🎏 풍향 표시 | 화살표로 시각 피드백 제공  
| 📊 점수 UI | 현재 점수, 남은 화살 수 표시
